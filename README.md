# visitcookiepro

Cookie Banner template voor VCP (Cookie Banner Total).

## Beschrijving
Dit template bevat een Smarty (.tpl) template voor het tonen van een cookie banner
binnen Visit Cookie Pro / VCP omgevingen.

## Inhoud
- 123 VCP Cookie Banner.tpl > Smarty template voor cookie banner weergave

## Vereisten
- VCP / Cookie Banner Total
- Smarty template ondersteuning

## Installatie
1. Download het bestand `123 VCP Cookie Banner.tpl`
2. Plaats dit bestand in de juiste template map van uw VCP installatie
3. Activeer het template binnen de VCP configuratie

## Gebruik
Na activatie wordt de cookie banner automatisch weergegeven volgens de ingestelde VCP regels.
